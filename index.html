Account Balance Calculator

This is a Java application that calculates and displays the starting and ending monthly balances for a checking account and a savings account.

How to Run

  1. Clone the repository: 
    git clone https://github.com/vanshaggarwal5/vansh_aggarwal_account_balance_calculator.git
  2. Navigate to the src folder: 
    cd vansh_aggarwal_account_balance_calculator/src
  3. Compile the Java files: 
    javac *.java
  4. Run the application: 
    java AccountBalanceApp
